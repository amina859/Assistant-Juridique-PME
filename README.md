# Assistant-Juridique-PME OK MyPME est une API intelligente développée avec FastAPI qui permet de répondre à des questions juridiques à partir d’un système RAG (Retrieval-Augmented Generation). Le projet utilise des documents PDF juridiques (droit OHADA et droit du travail sénégalais) qui sont chargés, découpés en segments et transformés en embeddings grâce à HuggingFace, puis stockés dans une base vectorielle FAISS pour effectuer une recherche sémantique. Lorsqu’un utilisateur pose une question via l’endpoint /chat, les passages les plus pertinents sont récupérés et envoyés à un modèle Mistral AI qui génère une réponse contextualisée et naturelle en français. L’application est configurée avec CORS, utilise un fichier .env pour la clé API, et sert d’assistant juridique destiné à accompagner les PME et entrepreneurs dans leurs démarches juridiques.
